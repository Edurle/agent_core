# agent_core

A general-purpose agent core library — the foundational layer for building AI agents.

## Status

🚧 Early stage — project scaffolding.

## Overview

`agent_core` is designed as a reusable **bottom-layer library** that provides core
capabilities for building AI agents:

- LLM interaction abstractions
- Tool / function calling framework
- Memory & context management
- Planning & execution loop primitives

> Design docs and architecture notes are maintained in the project's Obsidian vault,
> not in this repository.
